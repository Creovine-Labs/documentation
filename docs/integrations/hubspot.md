# HubSpot Integration

Connect Lira to HubSpot CRM for contact management, deal tracking, pipeline updates, and call logging.

## Setup

1. Go to **Organization Settings → Integrations → HubSpot**
2. Click **Connect HubSpot**
3. Authorize via HubSpot's OAuth 2.0 flow
4. Map organization members to HubSpot users

## Capabilities

| Feature | Description |
|:---|:---|
| Contacts | Create, read, update contacts |
| Companies | Manage company records |
| Deals | Track deals through pipeline stages |
| Pipelines | List and manage sales pipelines |
| Notes | Log meeting notes and call summaries |

## Sales Coaching Integration

When used with Sales Coaching, Lira automatically:

- Updates the deal record after a sales call
- Logs call notes with key discussion points
- Adjusts deal stage based on conversation signals
- Creates follow-up tasks linked to the deal
