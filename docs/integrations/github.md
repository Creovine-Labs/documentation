# GitHub Integration

Connect Lira to GitHub for repository browsing, issue creation, pull request management, and code search.

## Setup

1. Go to **Organization Settings → Integrations → GitHub**
2. Click **Connect GitHub**
3. Authorize the Lira GitHub OAuth App
4. Select repositories to grant access to

## Capabilities

| Feature | Description |
|:---|:---|
| Browse repos | List and explore repository contents |
| Read files | Access source code and documentation |
| Create issues | Push meeting tasks as GitHub issues |
| Create PRs | Open pull requests programmatically |
| Search code | Find code across connected repositories |

## Task → Issue Flow

Meeting action items related to code or engineering can be pushed to GitHub as issues with labels, assignees, and milestone references.
